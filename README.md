A modern, responsive portfolio website with a fully functional backend contact form. Built with HTML, CSS, JavaScript, and Node.js/Express.

## 🚀 Live Demo

**Frontend + Backend:** [https://umair-portfolio.onrender.com](https://umair-portfolio.onrender.com)

**Backend API:** [https://umair-portfolio.onrender.com/api/health](https://umair-portfolio.onrender.com/api/health)

## ✨ Features

### Frontend
- ✅ Modern, responsive design
- ✅ Dark/Light theme toggle
- ✅ Smooth animations and transitions
- ✅ Interactive UI elements
- ✅ Mobile-friendly navigation
- ✅ Scroll animations
- ✅ Contact form with validation

### Backend
- ✅ Express.js REST API
- ✅ Contact form endpoint
- ✅ Message storage (file + memory)
- ✅ CORS enabled
- ✅ Health check endpoint
- ✅ Error handling
- ✅ Deployed on Render.com

## 🛠️ Technologies Used

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter, Space Grotesk)
- CSS Variables for theming
- CSS Grid & Flexbox

### Backend
- Node.js
- Express.js
- CORS middleware
- File System (fs) for storage
